# BookingSync::API

This gem allows Ruby developers to programmatically access https://www.bookingsync.com

## Installation

Add this line to your application's Gemfile:

    gem 'bookingsync-api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bookingsync-api
